# UX Ideas

Improvements inspired by the original Notational Velocity and nvALT.

- [x] **Search highlight in results** — Highlight matching text in the title and excerpt so the user sees *why* a result matched.
- [x] **Escape in search bar clears the query** — Escape is "back up one level": editor → search, search → clear. Shows all notes.
- [ ] **Tab from search to editor** — Natural keyboard flow: find a note, Tab to start editing the selected note.
- [ ] **Note count indicator** — Show how many notes match the current query (e.g. "3 of 12 Notes").
- [ ] **Word/character count in editor** — Subtle status line showing word count, inspired by nvALT.
- [x] **Search clears on note creation** — After creating a note, clear the search bar so the full note list is visible behind the new note.
